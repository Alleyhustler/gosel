Moon widget


You can add my cool ahh widget to your site by adding code below! You can change width and height of it too. It calculates and shows user's moon angle, phase, azimuth and altitude.

By default it's on transparent background, you can put it inside a div and add background to that div. You can also change color of text by adding &textColor=%23ffffff (hex color after %23) to the iframe url. And also link color by adding &linkColor=%23ffffff (hex color after %23) to the iframe url.

There's support for custom CSS too. Just add &styleUrl=URL to the iframe url.

It works without JS too!!!!! (won't update every second without it though)
